Capi2Name Documentation



username:
    Der Benutzername (username) ist der Loginname, dieser Name kann nur einmalig im System sein.
    Er sollte kurz und aussagegräftig sein, z.B. Nachname+2 Buchstaben vom Vorname
    Wichtig: Der Benutzername ist zwingend erforderlich!

First name:
    You can save the real first name here.

Last name:
    You can save the real last name here.

show configpage:
    You can set here, if the user can show/edit his configuration.

show callback function:
    With this option you can hide the menupoint 'CallBack' and hide it from the call stat.

show the following MSNs in the call stat:
    With this option the admin can set witch MSN in showed in the call stat.
    For display all calls on all MSN leave this field blank, otherwise write down the numbers.
    You can seperate the numbers with a comma. Like this: 1234,344556,45678

set how many rows are display in the stat:
    With this option you can set how many rows are displayed in the man call stat. The default value is 15.

show prefix col in the stat:
    With this option you can set if the prefix col (where the call in comming from) is displayed in the call stat.

show on witch MSN is the call comming:
    With this option you can set if the called MSN is displayed in the call stat.

show call type:
    With this option you can set if the incomming call type (like FAX/TELEPHON) is displayed in the call stat.

allow delete entries:
    With this option you can set if the user can delete entries in the database or not.

password:
    Set a password for this user. Withou an password the user can't login to capi2name and look after the calls.
