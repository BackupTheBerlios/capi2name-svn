Capi2Name Dokumentation



Benutzername:
    Der Benutzername (username) ist der Loginname, dieser Name kann nur einmalig im System sein.
    Er sollte kurz und aussagegräftig sein, z.B. Nachname+2 Buchstaben vom Vorname
    Wichtig: Der Benutzername ist zwingend erforderlich!

Name:
    Der Name ist der Vollstäniger Namen eines Benutzers also Vorname + Nachname. Dieser Name wird im System verwendet, wenn man Notizen eröffnet.
    Der Eintrag des Names ist optimal, nicht zwingend erforderlich!

Menüpunkt Konfigurationsmenü:
    Mit dieser Option kann der Verwalter von Capi2name entscheiden, ob der Benutzer selber seine Optionen ändern kann.

Menüpunkt Rückruf:
    Mit dieser Option kann man die Rückruffunktion aus der Menüleiste und die Spalte aus der Anrufstatistik verstecken.

Menüpunkt Noitz:
    Mit dieser Option kann man die Noitzfunktion aus der Menüleiste verstecken.

Zeige nur folgende MSN's:
    Mit dieser Option kann der Verwalter von Capi2name festlegen, das der Benutzer nur eine MSN in der Anrufstatistik sieht.
    z.B. Nur von seinem Nummer!
    Um alle Anrufe auf allen MSN's anzeigen zu lassen, Feld einfach leer lassen.
    Sonst mit Kommas getrennt.
    z.B. 9458919,33455

Anzahl der Zeilen in der Statistik:
    Mit dieser Option kann man festlgen, wieviele Zeilen (Anrufe) in der Anrufstatistik angezeigt werden sollen.
    Default: 20

Zeige Vorwahlbereicht in Statistik:
    Mit dieser Option kann man festlegen, ob der Bereich aus dem ein Anruf kommt angezeigt werden soll. Dieser Namen geht von der Vorwahlnummer aus.

Zeige auf welcher MSN angerufen wurde:
    Dies legt fest, ob die MSN-Nummer angezeigt wird (in der Anrufstatistik), auf der Angerufen wurde.

Zeige Typ des Anrufes:
    Diese Option legt fest, ob die Dienstkennung in der Anrufstatistik angezeigt wird. Unter Dienstkennung versteht sich: FAX/Telefon/Daten

Einträge löschen erlaubt:
    Mit dieser Option kann man auswählen, ob der Benutzer auch Anrufe aus der Statistik löschen darf.
    Default: Aus: Man kann die Statistik beeinflussen. Mit Vorsicht zu benützen!!!

Passwort:
    Nur mit einem Passwort bekommt man zugriff auf die Statistik von Capi2Name!
